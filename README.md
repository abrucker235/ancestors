# Coding Challenge

Write a function/method that returns a collection of ancestor names.

## Example

```json
{
    "name": "Aaron",
    "mother": {
      "name": "Becky",
      "mother": {
        "name": "Joanne"
      },
      "father": {
          "name": "Donald"
      }
    },
    "father": {
      "name": "Steve",
      "mother": {
        "name": "Patricia"
      },
      "father": {
        "name": "Charles"
      }
    }
}
```
